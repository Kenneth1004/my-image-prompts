## 角色设定

### Character Identity Lock（每段 prompt 独立完整携带以下全部信息）

**脸型五官**
- 椭圆/心形小脸，下巴线条柔和微尖，颧骨不明显
- **白皙透亮肌肤**，带有自然红晕（脸颊、鼻尖、耳垂），皮肤质感细腻如瓷
- **杏仁形深棕色眼睛**，大而圆润，瞳孔清澈透亮，高光眼神温柔内敛
- 眉毛自然柔和，眉尾略弯，不修整的野生眉感
- **睫毛长而自然卷翘**（上下睫毛），不浓密但分明
- **小巧挺直的鼻梁**，鼻头圆润微翘
- **小巧饱满的嘴唇**，唇峰分明，自然粉嫩唇色，不涂口红
- 耳朵小巧，耳垂圆润

**发型**
- **黑色长直发**，长度到**锁骨下方约10cm**（约背部中段）
- 发质**丝滑有光泽**，如丝绸般顺滑
- **轻盈空气刘海**，长度到眉毛下方，不厚重，隐约露出额头
- 发尾自然内扣，不外翘
- 头发中分或偏分，自然垂落
- 阳光下带有深棕色光泽

**身高体型**
- **身高 162cm**
- **体型纤细匀称**，骨架小巧
- **肩窄腰细**，手臂纤长
- **手指修长白皙**，指甲修剪整齐不涂甲油
- 姿态优雅，站立时微微内八
- 坐姿端正，习惯性低头

**气质表情**
- **安静内敛**，不张扬，眼神温柔而略带忧郁
- 习惯性**微微低头**，目光向下或看向远方
- 表情层次丰富：行走时若有所思、等待时安静、眺望时略带寂寞
- **微笑时嘴角微微上扬**，不露齿，眼睛微弯
- 有一种**易碎的美感**，像玻璃窗上的雨滴

**服装细节**

| 单品 | 精确规格 |
|------|----------|
| 上装 | **白色水手服上衣**，面料为**挺括的棉质混纺**，海军蓝水手领，领口有**蓝白条纹领巾**，领巾系成蝴蝶结垂在胸前 |
| 下装 | **海军蓝百褶裙**，面料为**褶皱持久的聚酯纤维**，**裙长28cm**，褶皱整齐锋利，裙摆在膝盖上方约15cm |
| 袜子 | **白色过膝袜**，**20D厚度**，面料为**天鹅绒质感的尼龙**，袜口到**大腿中段**，有轻微的**蝴蝶结或蕾丝边装饰**，袜子略微透出肤色 |
| 鞋子 | **黑色乐福鞋**，**小牛皮**材质，鞋头圆润，**2cm低跟**，鞋面有金属马衔扣装饰 |
| 内搭 | 白色棉质**吊带背心**，不外露 |

**配饰**
- **黑色帆布托特包**，包带可调节，包身有简单的白色缝线装饰
- **白色无线耳机**（AirPods或类似），偶尔单耳佩戴
- **细发箍**或**小巧金属发夹**（不佩戴时夹在书包带上）
- **小巧耳钉**：银色或珍珠，每耳一枚
- 手腕偶尔戴**简约皮带手表**，表盘小巧

**随身物品**
- 文库本大小的**平装书**（封面素雅，有折角痕迹）
- **笔记本和笔**（黑色圆珠笔或自动铅笔）
- **折叠伞**（透明或淡蓝色，雨天场景）
- **学生证/交通卡**（偶尔露出）

### Style Lock（摄影风格锁定）

**相机系统**
- **Sony全画幅微单**（α7系列/α7R系列/α7C系列）
- **镜头**：Sony FE系列定焦镜头（如FE 55mm F1.8 ZA、FE 85mm F1.4 GM、FE 35mm F1.4 GM等）
- **色彩科学**：Sony自然色彩，肤色还原准确，不过度饱和

**光线**
- **柔和自然光为主**：窗边散射光、阴天柔光、傍晚金色时刻
- **避免硬光和强烈阴影**
- 偶尔有**逆光光晕**效果（头发边缘发光）

**色调**
- **低饱和度**，淡雅色彩
- 皮肤呈现**自然黄白皮质感**，不过白不过黄
- Sony特有的**柔和肤色渲染**

**构图**
- **中景和近景特写为主**，偶尔全身
- 注重**手部细节**和**表情层次**
- **浅景深**，背景虚化柔和

**画幅**
- **统一2:1画幅**，电影感宽幅构图

**氛围**
- **安静、内敛、略带忧郁的诗意感**
- 轻微胶片颗粒质感
- 日系电影美学（是枝裕和、岩井俊二风格）

### Continuity Lock（跨场景一致性）

**严格不变**
- 黑色长直发 + 空气刘海 + 发尾内扣 + 长度到锁骨下方
- 白皙透亮肌肤 + 自然红晕
- 杏仁形深棕色眼睛 + 温柔内敛眼神
- 白色水手服上衣 + 蓝白条纹领巾 + 海军蓝百褶裙（28cm）+ 白色20D过膝袜 + 黑色乐福鞋
- 黑色帆布托特包 + 白色无线耳机
- 纤细匀称体型 + 优雅姿态

**可变**
- 姿势（行走/站立/倚靠/蹲坐）
- 表情（若有所思/安静/微笑/忧郁）
- 视角（平拍/轻微俯拍/仰拍）
- 光线方向和强度
- 配饰佩戴（耳机单耳/双耳/不戴，发夹戴/不戴）

**禁止**
- 不得改变发色和发型（黑色长直发+空气刘海）
- 不得改变瞳色（深棕色）
- 不得改变服装结构（水手服+百褶裙）
- 不得添加纹身、染发、夸张配饰
